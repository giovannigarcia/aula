Biblioteca que estabelece conceitos de baralho, cartas e monte de descarte,
que serão utilizados para implementar qualquer aplicação que trabalhe com jogo de cartas.

A biblioteca disponibiliza funcionalidades:
- embaralhar as cartas;
- cortar o baralho;
- retirar uma carta do início do baralho;
- retirar uma carta do fim do baralho;
- por uma carta do início do baralho no fim do baralho;
- retirar uma carta início do monte de descartes;
- inserir uma carta no início do monte de descartes;
- mostrar todas as cartas do monte de descarte,

Para a construção da aplicação é necessária a ferramenta ant
- No diretório src execute o comando: ant
